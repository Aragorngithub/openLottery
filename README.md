# 基于 Vite 构建的 Vue 3 项目。


## 环境依赖

**node 20+**,
**Vue 3.4.23 (2024年4月16日发布的最新稳定版本)**,
**Vite 代替 Vue CLI/webpack**
**Pinia 代替 VueX**
**element-plus 代替 element-ui**
**Vue i18n 升级到最新版本**
**Vue router 升级到最新版本**



## 部署步骤

**npm install**
**npm run dev**
**npm run build**
